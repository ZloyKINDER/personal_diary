# Личный дневник

Веб-приложение для ведения личных записей с возможностью поиска и управления пользователями.

## Технологии

- Python 3.11
- Django 4.2
- PostgreSQL 15
- Bootstrap 5
- Docker / Docker Compose
- Poetry (управление зависимостями)

## Быстрый старт

### Требования

- Установленный Docker и Docker Compose
- Установленный Poetry (опционально, если запускаете без контейнера)

### 1. Клонирование репозитория

```bash
git clone https://github.com/ZloyKINDER/personal_diary.git
cd personal_diary
```

### 2. Настройка окружения

```bash
cp .env.sample .env
```

### 3. Запуск с помощью Docker

```bash
docker compose up -d --build
```

### 4. Создание суперпользователя (для админки)

```bash
docker compose exec web poetry run python manage.py createsuperuser
```
Для входа в админ-панель и управления записями/пользователями: http://localhost:8000/admin/

## Структура проекта

```text
personal_diary/
├── config/             # Основные настройки Django
├── users/              # Приложение для пользователей (регистрация, авторизация)
├── entries/            # Приложение для записей дневника (CRUD, поиск)
├── templates/          # Общие HTML шаблоны (base.html)
├── static/             # Статические файлы (CSS, JS, изображения)
├── docker-compose.yml  # Конфигурация контейнеров
├── Dockerfile          # Инструкция для сборки образа
├── pyproject.toml      # Зависимости Poetry
├── .env                # Переменные окружения (не в Git)
└── manage.py           # Управляющий скрипт Django```
```

## Основные функции
1. Регистрация и аутентификация: Пользователи могут создавать аккаунты и входить в систему.

2. Личный дневник: Создание, редактирование и удаление записей. Каждый пользователь видит только свои записи.

3. Поиск: Быстрый поиск по заголовку или содержимому записей.

4. Адаптивный дизайн: Интерфейс на Bootstrap 5 отлично выглядит на любых устройствах.

5. Админка: Управление пользователями и записями через стандартную панель администратора Django.
